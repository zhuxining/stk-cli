# 报告模板

---

## 模板 1: 市场总览

```markdown
# 市场总览 — {YYYY-MM-DD}

## 一、市场概况

{盘面总结}

| 市场 | 指数 | 最新 | 涨跌幅 | 温度 | 等级 |
|------|------|------|--------|------|------|
| ... |

## 二、重要资讯

1. **{标题}** — {影响分析}
2. **{标题}** — {影响分析}
3. **{标题}** — {影响分析}

**关注板块/概念**: {从新闻中提炼}
```

---

## 模板 2: 技术热点

```markdown
# 技术热点 — {YYYY-MM-DD}

## 一、多空情绪判断

{基于 industries 的 bull_count/bear_count 判断整体情绪}

## 二、热点行业

| 行业 | 多方信号 | 空方信号 | 判断 |
|------|---------|---------|------|
| ... |

## 三、候选精选 (Top 15)

| # | 代码 | 名称 | 多方信号 | 空方冲突 | 评分 | 涨跌% | PE | 资金流(万) | 关键信号 |
|---|------|------|---------|---------|------|--------|-----|-----------|---------|
| ... |

## 四、已创建自选组

分组名: **{MMDD}**，包含以上 15 只标的。
后续跟踪: `stk watchlist scan {MMDD}` 或 `stk watchlist kline {MMDD}`
```

---

## 模板 3: 个股分析

```markdown
# 个股分析 — {symbol} {name} — {YYYY-MM-DD}

## 一、结论与建议

- **方向**: {看多/看空/中性} — {核心依据，1-2句}
- **操作建议**: {建议，附止损止盈价位及盈亏比}
- **风险提示**: {关键风险}

## 二、数据概览

| 指标 | 数值 |
|------|------|
| 评分 | **{score}** |
| 最新价 | {last}（{change_pct}%） |
| 估值 | PE {pe_ttm} / PB {pb} / 股息率 {dividend_yield}% |
| 趋势 | ADX {adx} |
| 风控 | 止损 {stop_loss} / 止盈 {take_profit} / 盈亏比 {risk_reward_ratio} |
| 资金 | {capital_flow} 万 |

**评分明细:** {score_detail 各维度信号}

**技术面:**
- 均线/MACD/RSI/BOLL/量价 — 综合解读，聚焦关键变化

## 三、行业对比

{成长性/估值/杜邦 对比表格，标注目标股相对行业均值的优劣势}

## 四、行业推荐

{如行业对比中发现同业优秀标的（成长性好、估值合理），推荐关注并简述理由；无则省略此节}
```

---

## 模板 4: 分组整体分析

```markdown
# 分组分析 — {group_name} — {YYYY-MM-DD}

## 一、逐只点评

按评分降序，每只含：排名、评分、涨跌%、关键信号、技术展望、操作建议。
signals 中含 `[买]`/`[卖]`/`[警]` 前缀的标的额外标注异动类型。

### 1. {symbol} {name} — 评分 {score} | {change_pct}%

- **信号**: {关键信号}
- **展望**: {方向} — {依据}
- **建议**: {操作建议}
{如有异动: - **⚠ 异动**: {类型 + 详情}}

（逐只展开...）

## 二、整体建议

- **重点关注**: {Top 3 及理由}
- **风险提示**: {整体风险}
```
