# 报告模板

## 选股报告

```
# 选股报告 - {日期}

## 策略: {策略名称}

## 市场环境
- 大盘温度: {temp}/100
- 市场宽度: 涨{up} / 跌{down} / 平{flat}
- 主要指数: ...

## 板块方向
| 板块 | 涨跌幅 | 资金流向 |
|------|--------|---------|

## 板块个股筛选
### {板块名称1}
| 代码 | 名称 | 涨跌幅 | 资金净流入 | 入选 |
|------|------|--------|-----------|------|
(board detail 按资金净流入排序，标注前3-5只)

### {板块名称2}
(同上结构)

板块候选池合并: {去重后的股票列表}

## 三维度交叉验证
| 代码 | 名称 | 板块候选 | 技术候选 | 资金候选 | 命中维度 |
|------|------|---------|---------|---------|---------|

## 候选股票池
| 代码 | 名称 | 现价 | 涨跌幅 | 入选理由 |
|------|------|------|--------|---------|

## 重点关注 (Top 3)
### {代码} {名称}
- 技术面: ...
- 资金面: ...
- 基本面: ...
```

## 扫描报告

```
# 扫描报告 - {日期}

## 一、大盘概览

| 指数 | 收盘 | 涨跌幅 |
|------|------|--------|
| 上证指数 | {close} | {pct}% |
| 深证成指 | {close} | {pct}% |
| 创业板指 | {close} | {pct}% |

- 市场温度: {temp}/100 ({等级})
- 涨跌统计: 上涨 {up} / 下跌 {down} / 涨停 {limit_up} / 跌停 {limit_down}

## 二、板块热点

### 行业涨幅前5
| 排名 | 板块 | 涨跌幅 | 资金净流入 |
|------|------|--------|-----------|

### 热门概念
{concept1}, {concept2}, {concept3}

### 资金持续性验证
{板块资金流入天数与趋势}

## 三、资讯要点
1. {新闻标题} — {简评}
2. ...

## 四、自选股深度扫描

### {代码} {名称}

| 指标 | 数值 | 信号 |
|------|------|------|
| 收盘价 | {price} | 涨跌 {pct}% |
| 资金流向 | {net_flow} | {流入/流出} |
| MACD | {DIF}/{DEA} | {金叉/死叉/多头/空头} |
| RSI | {rsi} | {超买/正常/超卖} |
| KDJ | K:{k} D:{d} J:{j} | {信号} |
| BOLL | 上:{upper} 中:{mid} 下:{lower} | 当前位于{位置} |
| 筹码 | 获利比例 {pct}% | 套牢盘 {pct}% |

K线走势(近10日): {趋势描述}

综合评价:
- 技术展望: {看多/看空/中性} — {依据}
- 风险等级: {低/中/高}
- 操作建议: {持有/减仓/加仓} — {理由}

---
(每只自选股重复以上结构)

## 五、明日关注
| 股票 | 支撑位 | 压力位 | 关注事项 |
|------|--------|--------|---------|

关注板块: {板块1}, {板块2}
风险提示: {需警惕的事项}
```

## 盯盘报告

```
# 盯盘报告 - {日期} {时间}

## 市场环境速览
- 指数: 上证 {value}({pct}%) / 深证 {value}({pct}%) / 创业板 {value}({pct}%)
- 温度: {temp}/100

## 异动预警
| 股票 | 预警类型 | 详情 | 建议 |
|------|---------|------|------|

## 持仓逐一检查
### {代码} {名称}
- 现价: {price} | 涨跌: {change}%
- 资金: 主力{main_flow} | 散户{retail_flow}
- 技术信号: MACD({状态}) RSI({值}) KDJ({状态})
- 建议: {操作建议}

---
(每只持仓重复以上结构)
```

```
| 周一 | {temp} | {index} | {up}/{down} | {event} |
| 周二 | {temp} | {index} | {up}/{down} | {event} |
| ... | | | | |

趋势判断: {升温/降温/震荡}，关键转折点在{日期}

## 二、板块轮动追踪

持续强势: {板块列表} — 连续{N}日领涨
由强转弱: {板块列表} — {转弱原因}
新崛起: {板块列表} — {催化因素}

| 板块 | 周一排名 | 周二排名 | ... | 周五排名 | 趋势 |
|------|---------|---------|-----|---------|------|

## 三、自选股周表现

| 股票 | 周初价 | 周末价 | 周涨跌 | 关键转折 |
|------|--------|--------|--------|---------|

## 四、策略回顾

| 推荐日期 | 股票 | 推荐价 | 当前价 | 收益 | 评价 |
|---------|------|--------|--------|------|------|

### 预警验证
| 预警日期 | 股票 | 预警内容 | 后续走势 | 是否应验 |
|---------|------|---------|---------|---------|

### 操作建议执行效果
{对本周加仓/减仓建议的回顾}

## 五、总结与下周计划

### 本周主线
{一段话总结本周市场主线逻辑}

### 持仓调整建议
- 加仓: {股票} — {理由}
- 减仓: {股票} — {理由}
- 换仓: {卖出} -> {买入} — {理由}

### 下周关注
- 板块: {关注方向}
- 个股: {重点观察标的}
- 事件: {重要日程/数据发布}
- 风险: {需规避的风险点}
```
